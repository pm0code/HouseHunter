## Decisions Made This Session (with rationale)

| Decision | Rationale | Evidence |
|---|---|---|
| MapLibre GL JS v5.24 (not Mapbox GL JS) | Mapbox GL JS v2+ is proprietary/metered ($5/1K loads); MapLibre is BSD-3, AWS/MS-backed, drop-in for all 4 needed features | mapbox.com/pricing; github.com/maplibre |
| MapTiler tiles (not Mapbox tiles) | Decouples from Mapbox entirely; 100K free/month | maptiler.com/pricing |
| Mapbox Directions pre-computed at ingest | Reduces API calls from ~75K/month to ~400/month; walk time is stable data | PRD §6.2 ingest pattern |
| Drizzle ORM | Best TypeScript + PostGIS balance; Prisma eliminated (geometry unsupported wall) | orm.drizzle.team; GitHub 34.5k stars |
| TypeScript throughout | Next.js 15 default; both map libs have first-party types | nextjs.org/docs |
| Tailwind + shadcn/ui | Zero-runtime-dependency, all required components, WCAG-compliant Radix primitives | ui.shadcn.com/docs |
| nuqs + Zustand + TanStack Query | Three state domains require three different solutions (URL/ephemeral/server) | Next.js Conf 2025; tanstack.com |
| Geocodio for batch ingest geocoding | Mapbox "temporary" tier prohibits storing results; Geocodio allows it at $1/1K | geocod.io pricing |
| data.ny.gov Socrata (not MTA CSV) | Structured API, SoQL filterable, 170 Brooklyn stations | data.ny.gov/5f5g-n3cz |
| Supabase Pro + Railway for deployment | Supabase = lowest-friction PostGIS; Railway = cron + Redis + web service in one | railway.com; supabase.com |
